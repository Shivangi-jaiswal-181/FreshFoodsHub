# FreshFoodsHub Documentation

## Overview
FreshFoodsHub is an online marketplace for locally processed food products.

## Features
- User Registration & Login
- Product Listing
- Product Search & Browse
- Online Ordering & Payment
- Order Tracking

## AI Feature
AI-based Product Recommendation System

## Tech Stack
- Next.js
- Node.js
- Express.js
- MongoDB
- JWT/Firebase Auth
- Razorpay/Stripe
